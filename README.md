# ProjectLogs

## サービス概要
チームでプロジェクトを管理し、タスクごとの作業時間を記録できるシンプルなWebアプリケーションです。  
Laravelを使用して開発され、招待制のアクセス管理や作業時間の集計・可視化機能を提供します。
